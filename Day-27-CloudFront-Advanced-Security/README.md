# Day 35 – CloudFront Advanced Security

## Project Goal

To secure global content delivery using CloudFront with a private S3 origin, Origin Access Control, geo-restrictions, signed URLs, and field-level encryption.

## Services Used

- Amazon CloudFront
- Amazon S3
- Origin Access Control
- CloudFront Signed URLs
- Field-Level Encryption

## Outcome

A secure edge delivery architecture was implemented, protecting private S3 content behind CloudFront while enforcing controlled global access and advanced content protection mechanisms.

## Key Learning

- CloudFront can securely serve private S3 content.
- OAC is preferred for modern CloudFront-to-S3 origin access.
- Signed URLs protect premium/private content.
- Geo-restriction supports regional access control.
