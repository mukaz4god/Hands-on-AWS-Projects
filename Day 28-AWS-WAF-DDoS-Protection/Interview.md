# Day 36 – AWS WAF & DDoS Protection

# Interview Ready Wording

## Question: What is AWS WAF used for?

**Answer:**  
AWS WAF protects web applications by inspecting HTTP and HTTPS requests and applying rules to allow, block, count, or rate-limit traffic.

---

## Question: What is the difference between AWS WAF and AWS Shield?

**Answer:**  
AWS WAF focuses on application-layer web request filtering, while AWS Shield provides DDoS protection. Shield Standard is automatically available, while Shield Advanced provides enhanced protection and response support.

---

## Question: Why use AWS Managed Rules?

**Answer:**  
AWS Managed Rules provide ready-made protections against common attack patterns such as known bad inputs, SQL injection, and common web exploit attempts.

---

## Question: Why use rate-based rules?

**Answer:**  
Rate-based rules reduce abuse by blocking or counting clients that exceed a request threshold, which helps mitigate bots, scraping, brute-force attempts, and request floods.
