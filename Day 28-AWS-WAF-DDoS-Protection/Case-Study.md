# AWS WAF & DDoS Protection

## Case Study – Protecting a Public Web Application from Malicious Traffic

### Business Scenario

A SaaS company hosts a public web application behind an Application Load Balancer and CloudFront. The application receives suspicious traffic spikes, bot-like requests, SQL injection attempts, and repeated requests from single IP addresses.

The security team wants to introduce a Web Application Firewall and DDoS-aware protection.

### Security Requirements

- Create a Web ACL.
- Add AWS Managed Rules.
- Add rate limiting.
- Enable logging.
- Understand AWS Shield protection.
- Simulate malicious traffic safely.

### Your Role

You are the Cloud Security Engineer responsible for protecting the web application at the edge and application layer.

### Project Goal

Deploy AWS WAF protections to inspect HTTP/S traffic, block common attacks, reduce abusive request patterns, and improve web security visibility.
