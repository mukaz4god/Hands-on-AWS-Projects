# Day 36 – AWS WAF & DDoS Protection

## Project Goal

To protect a public web application using AWS WAF, AWS Managed Rules, rate limiting, logging, and AWS Shield-aware DDoS protection.

## Services Used

- AWS WAF
- AWS Managed Rules
- AWS Shield
- CloudWatch Logs
- CloudFront / ALB / API Gateway

## Outcome

A web application firewall solution was deployed, enabling inspection and control of malicious traffic, request-rate abuse protection, and improved visibility into web threats.

## Key Learning

- WAF protects Layer 7 HTTP/S traffic.
- Managed rules provide quick security coverage.
- Rate-based rules reduce abuse.
- Logging is essential for tuning and investigations.
