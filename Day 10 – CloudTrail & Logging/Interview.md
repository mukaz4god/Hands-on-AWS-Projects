# Interview Ready Wording

**Q: What is CloudTrail used for?**<br/>
**Answer:** CloudTrail records all API activity in AWS, providing visibility into user actions, resource changes, and system events for auditing and security monitoring.

**Q: Why enable multi-region trails?**<br/>
**Answer:** Multi-region trails ensure all activity across AWS regions is logged, preventing blind spots in monitoring and improving compliance coverage.

**Q: Why use Athena with CloudTrail?**<br/>
**Answer:** Athena allows querying CloudTrail logs using SQL, enabling efficient investigation of security incidents and user activity.
