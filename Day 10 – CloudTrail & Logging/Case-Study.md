# Case Study – Audit Logging and Compliance in AWS

**Business Scenario**<br/>
KFJ Solutions, a regulated company must track all activities performed in their AWS environment for security auditing and compliance (ISO 27001, SOC 2). They require:<br/>
- Visibility into API calls and user activity
- Centralized log storage
- Tamper-proof logging
- Ability to query logs for investigations

 **Your Task**<br/>
You are the Cloud Security Engineer responsible for implementing:
- AWS CloudTrail (multi-region)
- Centralized S3 logging
- Log validation
- Querying logs using Athena
