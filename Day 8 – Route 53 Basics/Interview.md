## Interview Ready Wording
**Q: What is Route 53?**
**Answer:** Route 53 is AWS’s DNS service used to route user traffic to AWS resources such as ALBs, EC2, and S3 using domain names.

**Q: What is an Alias record?**
**Answer:** An Alias record maps a domain directly to AWS resources like an ALB without requiring an IP address, unlike traditional A records.

**Q: Why use health checks?**
**Answer:** Health checks allow Route 53 to monitor endpoint availability and automatically route traffic away from unhealthy resources, improving reliability.
