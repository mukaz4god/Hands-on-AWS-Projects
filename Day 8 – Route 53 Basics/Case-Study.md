## Case Study – DNS Management for a Web Application
**Business Scenario**<br/>

A company has deployed its application using an ALB but users must access it via a long AWS-generated DNS name.

To improve usability and reliability, the company wants:
- A custom domain (e.g., example.com)
- DNS routing using Route 53
- Health checks and failover

**Your Task:**
You are the Cloud Engineer responsible for:
- Configuring DNS using Route 53
- Mapping domain to ALB
- Implementing health checks
