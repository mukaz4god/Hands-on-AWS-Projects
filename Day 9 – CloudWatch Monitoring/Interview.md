# Interview Ready Wording

**Q: What is CloudWatch used for?** <br/>
**Answer:** CloudWatch is used for monitoring AWS resources and applications by collecting metrics, logs, and events, and triggering alarms when thresholds are exceeded.

**Q: Why enable detailed monitoring?**<br/>
**Answer:** Detailed monitoring provides higher granularity metrics (1-minute intervals), which improves visibility and faster detection of issues compared to default 5-minute monitoring.

**Q: How do you implement alerting in AWS?**<br/>
**Answer:** By creating CloudWatch alarms based on metrics and integrating them with SNS topics to send notifications via email, SMS, or other endpoints.
