# AWS Network Firewall

## Case Study – Centralized VPC Traffic Inspection

### Business Scenario

A company wants deeper network inspection than Security Groups and NACLs provide. The security team needs centralized firewall controls that can inspect traffic entering, leaving, or moving across VPC segments.

The company decides to deploy AWS Network Firewall with stateless and stateful rule groups.

### Security Requirements

- Deploy AWS Network Firewall.
- Create stateless rules.
- Create stateful rules.
- Create firewall policy.
- Route traffic through firewall endpoints.
- Test traffic inspection.

### Your Role

You are the Network Security Engineer responsible for deploying managed firewall inspection inside the VPC.

### Project Goal

Implement centralized VPC traffic inspection using AWS Network Firewall with stateless and stateful filtering.
